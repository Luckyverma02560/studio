# **App Name**: Demystify Capital Markets Clone

## Core Features:

- Exact Page Replication: Replicate the entire "About Us" page, ensuring every element is a pixel-perfect match, including layout, content, and styling.
- Content Mirroring: Duplicate all text, images, and multimedia content from the original page without modification.
- Theme and Style Cloning: Clone the identical visual theme: the specific colors, font, background and the overall visual identity of the website.
- Animation Duplication: Recreate all animations and interactive elements to function identically to the original site.

## Style Guidelines:

- Primary color: Dark grey (#222222) to emulate a sophisticated, professional feel.
- Background color: Off-white (#F5F5F5) for a clean and readable content area.
- Accent color: Gold (#FFD700) to highlight key sections and maintain visual consistency.
- Body and headline font: Implement 'PT Sans' (sans-serif) for consistent readability and modern aesthetics throughout the site.
- Maintain the same icons as original
- Clone the original layout, including sections and spacing.
- Use the same subtle fade-in animations as the original site.
